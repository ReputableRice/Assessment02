## Description
The main theme about this app is showcasing different fruits, providing a succinct explanations and a brief on their history. 

The values each bit represents
- bit 1 = apple
- bit 2 = banana
- bit 4 = blueberry
- bit 8 = grape
- bit 16 = melon
- bit 32 = orange
